Welcome to the drachtio media resouree function, empowering nodejs/javascript developers to build rich media application on SIP / VoIP networks

drachtio is an open-source, nodejs-based ecosystem for creating any kind of VoIP server-based application: registrar, proxy, back-to-back user agent, and many others. Furthermore, when coupled with the drachtio media resource function, rich media-processing applications can be easily built as well.

Within the drachtio ecosystem, drachtio-fsmrf is a high-level media processing abstraction framework that integrates with a freeswitch media server and partners with the drachtio signaling resource framework.
